# 仕入れ判断ルール

## 同一商品判断

- eBay画像・タイトルと照合
- 色違い・サイズ違い → **別商品**
- セット商品のバラ売り → **対象外**

---

## 価格条件

- F列の希望価格以下（税込み・送料込み）
- 送料不明時 → **+1300円で計算**
- ポイント還元は考慮しない

### オークション判断

- 即決商品優先
- 現在価格が希望価格内ならOK（即決あればそちら優先）
- 入札中商品も対象

---

## コンディション条件

### P列による判断

| P列の値 | 判断 |
|---------|------|
| 「中古」or 空 | 新品・中古どちらもOK |
| 「新品」 | 新品優先、なければ中古可（W列に記載） |

### 表記の扱い

| 表記 | 扱い |
|------|------|
| 「未使用に近い」 | 新品扱い |
| 「目立った傷や汚れなし」 | 中古扱い |
| 「難あり」「訳あり」「破損」「欠品」「ジャンク」 | **除外** |
| 「箱なし」「説明書なし」 | 他に候補なければ適用可 |
| 売り切れ商品 | **除外** |
